# Репозиторий: Каноническая структура

Цель: оставить ядро неизменным, а вспомогательные артефакты вынести в логичные каталоги без изменения логики.

- /app — основное ядро (бот, сервисы, хранилище)
- /entrypoints — точки входа (тонкие файлы запуска)
- /tools — аудит, автофиксы, проверки, генераторы отчетов (скрипты)
- /scripts — служебные скрипты для CI/операций (оставляем необходимые)
- /docs — документация, руководства, отчеты в виде Markdown
- /archive — старые реализации и снапшоты (например, 5656-main, legacy main_render.py)

## Запуск инструментов из /tools
Все перемещенные утилиты работают как раньше. Запускать из корня:

```
python tools/auto_fix_all.py
python tools/check_callback_data.py
python tools/run_all_final_checks.py
```

Если скрипт импортирует модули из корня, это поддерживается автоматически (корень добавлен в `sys.path`).
